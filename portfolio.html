Selected Work

Tool Tracker

A simple app to view tool usage by user or system and record new assignments. Focused on clean data model, predictable API endpoints, and fast lookup.

  • Built CRUD endpoints and basic auth.
  • Wrote queries to surface usage by user/tool.
  • Added import/export so admins can review offline.

Technologies: Node.js, Express, PostgreSQL

Meeting Minutes Generator

Turns transcripts and attendee lists into structured minutes (HTML/PDF) using prompts and templates; reduces manual editing.

  • Parsed inputs, segmented speakers, and generated sections (purpose, decisions, action items).
  • Produced downloadable HTML/PDF output.
  • Added lightweight review step for final edits.

Technologies: Python, Power Automate, Power BI

PDF → Table Extractor

Extracts fields from PDFs and normalizes them into clean tables for analysis and export.

  • Used OCR for text capture, then rules to map to columns.
  • Handled noisy inputs and partial fields gracefully.
  • Exported CSV for downstream tools.

Technologies: Python, Azure OCR, FastAPI

Technologies: React, Node.js, Express, MongoDB

SkyScore
